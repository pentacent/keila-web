# API: Integrate Email Newsletters Into Your Existing Apps

You can use the Keila API to integrate your email newsletters with your existing
apps, and programmatically manage your contacts and campaigns.

The API design is based on [JSON API](https://jsonapi.org/), making it easy to
understand and work with.

## Authentication

The API uses Bearer authentication with a private API key. See
[Authentication](/docs/api/authentication) for how to create an API key and how
to use it in a request.

## API Reference

The API reference is organized by resource:

- [Campaigns](/docs/api/campaigns)
- [Contacts](/docs/api/contacts)
- [Forms](/docs/api/forms)
- [Segments](/docs/api/segments)
- [Senders](/docs/api/senders)
