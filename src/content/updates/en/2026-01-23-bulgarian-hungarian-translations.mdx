Thanks to community contributions, the Keila interface is now fully available in both Bulgarian and Hungarian.
This brings the total number of supported languages to six: English, German, French, Spanish, Bulgarian, and Hungarian.

Благодарим ви and köszönjük to everyone who helped make these translations possible!

Want to use Keila in your own language? There is a short [technical translation guide on GitHub](https://github.com/pentacent/keila/blob/main/CONTRIBUTING.md#translating-keila).
If you would like to contribute and need help getting started, please don't hesitate to reach out on GitHub
